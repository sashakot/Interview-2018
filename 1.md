
### Write a function to swap variables in place whithout addtiona variable

``` C
// For integers only. Overflow
void swap(int a, int b)
{
  a = a + b;
  b = a - b;
  a = a - b; 
}

// Any type
void swap(t a, t b)
{
  a = a ^ b;
  b = a ^ b;
  a = a ^ b; 
}
```
